🔍 Manual DevTools Debugging Test

Open Chrome DevTools → Sources panel to verify:

  1. Ensure scheme-debug=true is in the URL or a script tag has the debug attribute.
  2. Look in the file tree for scheme://inline/script-0.scm and scheme://app/.../manual_script.scm.
  3. Set a breakpoint and refresh — execution should pause, including on recursive calls. ✅

The "Scheme Stack" sidebar (extension) shows call stack and variables when paused. Step Into / Over / Out buttons use interpreter-managed stepping.

Execution results will appear in the DevTools console.
